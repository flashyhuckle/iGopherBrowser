iGopherBrowser

A modern gopher client for devices

Gopher might be a protocol from the past, but it is still alive and well. iGopherBrowser is a modern gopher client for iOS and macOS, which lets you browse the gopherspace with ease.

Built on top of the swift-gopher library, iGopherBrowser is a new way to browse the gopherspace.

  • Supports original RFC 1436
  • Support for major gopher types, along with preview support through QuickLook
  • Hierarchical view of gopher menus (macOS only)
  • Full accessibility support
  • Clean, modern UI
  • Supports all gopher search implementations
